Exit Criteria

Критерии выхода по фазам

Из роадмапа v11. Каждая фаза имеет чёткие метрики для перехода к следующему этапу.

Phase 0: Валидация (2 недели)

  • Golden Set ready: 20-25 договоров (4-5 на каждый из 5 типов)
  • Recall ≥ 70% на ручном прогоне
  • 5 чек-листов утверждены юристом пилота
  • 12 формулировок: DRAFT → VERIFIED

Этап 1: MVP (Q2 2026, 4 недели)

  • Recall ≥ 90% (critical), ≥ 80% (all)
  • Precision ≥ 80%
  • 5 типов договоров обрабатываются
  • Q&A по базе знаний работает
  • Track Changes обрабатывается
  • Inline-редактирование формулировок

Этап 2 (Q2-Q3 2026, 12 недель)

  • 12+ типов договоров
  • NPS ≥ 30
  • WAU ≥ 70% юристов пилота
  • Feedback loop работает

Этап 3 (Q4 2026+)

  • Интеграция с ≥1 СЭД
  • Генерация ≥3 типов документов
  • Скоринг MVP

Итого по всем AI-сотрудникам

18 юридических ролей в организации → 6 получают AI-замену в 2026, 7 — потенциал, 5 остаются за человеком.
По 6 заменяемым ролям: 63 функции описаны, из них покрыто роадмапом: 34 делаем, 6 частично, 23 не покрыто.
Самый высокий потенциал автоматизации: Document Comparison (75%) и Knowledge Q&A (70%).
Самый низкий: Скоринг исков (40%) — TBD, требует отдельного датасета судебной практики, высокий риск.
Ключевой AI-сотрудник: Цифровой юрист по договорам (LC-1 + LC-5) — анализ + составление, 30% юр. функции, MVP Q2 2026.
